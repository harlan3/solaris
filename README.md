# solaris
Distributed multicast transmission of JSON data structures

Solaris provides distributed multicast data transmission between Java and/or C++ processes using JSON data structures.

**Defaulting Java to use IPv4**  
java -Djava.net.preferIPv4Stack=true -jar dist\solaris_server.jar  
java -Djava.net.preferIPv4Stack=true -jar dist\solaris_client.jar
